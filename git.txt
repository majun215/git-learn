1.  git status
2.  git config --list:查看用户名和邮箱
3.  git add -A:往暂存区添加文件和内容
4.  git commit -m "注释和说明":提交文件
5.  git remote add origin https://github.com/majun215/APUE.git:远端仓库
6.  git push origin master:提交到远端仓库，上传到服务器
7.  git remote -v:查看远端仓库的地址和远端仓库的对应关系
8.  git clone https://github.com/majun215/APUE.git:下载到本地
9.  git pull origin master:拉取远端仓库中的更新
10. git clean -d -fx:清除本地文件
11. git config --global credential.helper store 提交时不用输邮箱账号密码
